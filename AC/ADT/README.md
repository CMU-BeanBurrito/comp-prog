To access current ADT: https://atcoder.jp/contests/adt_top

To access past ADT: https://atcoder.jp/contests/adt_[difficulty]_[yyyymmdd]_[timeslot]
- difficulty: easy, medium, hard, all
- yyyymmdd: year, month, day. contests are on Tuesdays, Wednesdays, Thursdays
- timeslot: 1, 2, 3
- remove brackets

See sub-directories within this folder for examples
